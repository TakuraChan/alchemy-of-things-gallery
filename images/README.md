## Image Requirements

### File Format:
- **JPG** (recommended for photographs)
- PNG if you need transparency

### Size:
- **Width/Height:** 1200-1600px on longest side
- **File Size:** Under 500KB per image
- **Aspect Ratio:** 4:5 or 3:4 works best (portrait or close to square)

### Compression:
Use these free tools to reduce file size without losing quality:
- **TinyPNG:** tinypng.com
- **Squoosh:** squoosh.app
- **Photoshop:** "Save for Web" feature

## Photography Tips

**For best results when photographing your paintings:**

1. **Lighting:**
   - Natural light (cloudy day is ideal)
   - Avoid direct sunlight (creates hotspots)
   - Use even, diffused lighting

2. **Setup:**
   - Hang painting flat on wall
   - Camera perpendicular to painting (straight on)
   - Use tripod if possible

3. **Camera Settings:**
   - Turn off flash
   - Use highest resolution
   - Take multiple shots

4. **Post-Processing:**
   - Crop to just the painting
   - Adjust brightness/contrast if needed
   - Don't over-edit colors
   - Keep true to original artwork
